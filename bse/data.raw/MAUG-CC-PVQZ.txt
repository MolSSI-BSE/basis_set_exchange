FROM DESCRIPTION:
----------------
maug-cc-pVQZ basis set is the cc-pVQZ basis set of Dunning and coworkers augmented with only s and p diffuse functions on the heavy atoms. This basis set is called minimally augmented cc-pVQZ, which we abbreviate maug-cc-pVQZ
